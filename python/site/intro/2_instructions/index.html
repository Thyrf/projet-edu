Commentaires et instructions simples

Pour chaque section de cette page, je partagerais un code qu'il vous faudra recopier dans un interpreteur python. Pour cela vous pouvez installer python sur votre ordinateur ainsi qu'un editeur de code, mais pour débuter il est plus simple d'utiliser un environnement de développement tel que Thonny. Si vous ne souhaitez rien installer sur votre ordinateur, vous pouvez aussi utiliser des interpréteurs en ligne tels que Trinket que vous trouverez sur cette page.

Voici un programme. Lisez le code, essayez de prévoir ce qui va se passer à son execution et testez le sur l'interpreteur. 

  # Tout d'abord, les lignes commençant par un Hashtag sont des commentaires. Ceux-ci sont très pratiques pour rendre le code python compréhensible et le partager. Cela me sera très utile maintenant, comme vous pouvez l'imaginer.

  # Alors, que pouvons-nous faire en python ? Comme dans d'autres langages, nous pouvons donner des instructions. Commençons par la norme :
  print("Hello world")

Exercice 1

Exécutez ce code, changez le, changez le texte, est-ce print() marche sans les gillements ? Pour tester cela, vous pouvez aussi taper directement des instructions dans le terminal.
